# Monitor Electron App

Minimal Electron scaffold with separated `main`, `preload`, and `renderer` layers.

## Project structure

```text
.
├── package.json
└── src
    ├── main
    │   ├── main.js
    │   └── preload.js
    └── renderer
        ├── index.html
        ├── renderer.js
        └── styles.css
```

## Getting started

```bash
npm install
npm run start
```
